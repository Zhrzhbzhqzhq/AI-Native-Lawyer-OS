# LawDesk Golden Case 01

This directory stores the first golden case dataset for LawDesk V1 validation.

## Structure

- `manifest.json` records the dataset identity and expected output file map.
- `materials/` stores original case materials.
- `expected/` stores the expected structured LawDesk outputs for this case.

All JSON files are initialized with valid empty structures. Populate them only with deterministic case data suitable for repeatable validation.
