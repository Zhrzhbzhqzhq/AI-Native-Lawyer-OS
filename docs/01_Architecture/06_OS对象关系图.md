# 06_OS 对象关系图

> AI Native Lawyer OS 顶层对象关系图  
> 本文件用于冻结系统最高层架构关系。

---

# 一、核心结论

AI Native Lawyer OS 不是“AI 页面 + 案件管理”。

它的核心结构是：

Today 负责工作入口。  
Matter Workspace 负责案件承载。  
Workflow 负责推进工作。  
AI Capability 负责全程协助。  
Domain Model 负责数据沉淀。
---
