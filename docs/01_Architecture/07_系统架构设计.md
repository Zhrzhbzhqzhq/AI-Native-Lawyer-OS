# 07_系统架构设计

> AI Native Lawyer OS V1  
> 本文档定义 V1 最小可运行系统架构。  
> 原则：够用、清晰、可开发，不做过度工程化。

---

# 一、架构目标

V1 系统架构目标：

- 支撑独立律师本地使用
- 支撑案件工作区
- 支撑 13 条 Workflow
- 支撑 AI 全程参与
- 支撑文件上传与资料管理
- 支撑数据库持久化
- 支撑 Today 工作台
- 支撑后续扩展

V1 不追求大规模团队协作，不做律所 ERP。
